
# Enron Email Explorer

An AI-powered web application to analyze and explore the Enron Email Dataset.

## Features

- Email search and filtering
- Topic tagging for emails
- AI-generated summaries
- Narrative story generation
- Data visualization

## Technologies Used

- Python
- Flask
- Pandas
- NLP
- OpenAI API

## Project Structure

app.py → main Flask application  
enron_analysis.py → email analysis  
generate_summaries.py → email summarization  
generate_story_insight.py → narrative generation  

templates/ → HTML pages  
static/ → CSS and images  
utils/ → helper functions  

## Dataset

The dataset used in this project was provided as part of a university course and is not included in this repository.

## Run the Project

###Install dependencies

```bash
pip install -r requirements.txt
```


###Run the application

```bash
python app.py
```


Open browser

http://127.0.0.1:5000
